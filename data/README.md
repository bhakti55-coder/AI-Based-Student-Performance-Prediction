# Data
This folder contains raw and processed datasets used for the AI Based Student Performance Prediction project.

This directory contains the dataset used to train the AI model. 

* **`student_data.csv`**: The primary dataset containing 2,392 records and 15 features (e.g., StudyTimeWeekly, Absences, Tutoring, GPA). 

*Note: The dataset is fully cleaned with 0 missing values.*
